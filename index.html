Tentang Saya

Saya adalah seorang profesional yang bersemangat di bidang pengembangan software atau perangkat lunak seperti mobile apps atau aplikasi web, dengan sedikit pengalaman lebih di berbagai project baik tugas kuliah atau kesingan pribadi dalam menciptakan solusi yang efisien dan inovatif. Saya percaya bahwa teknologi dapat memberdayakan bisnis dan individu untuk mencapai potensi penuh mereka.

Fokus utama saya adalah pada pengembangan aplikasi web performa tinggi menggunakan framework modern seperti React dan Node.js ataupun juga laravel, serta mendalami analisis data kompleks menggunakan berbagai tools. Saya selalu bersemangat untuk mempelajari teknologi baru dan menghadapi tantangan yang kompleks.

Selain ngoding dan data, saya menjadi asisten praktikum di laboratorium informatika Universitas Ahmad Dahlan untuk mata kuliah seperti Algoritma Pemrograman, Kecerdasan Buatan, Grafika Komputer, Pemrogramana Berorientasi Objek(OOP), serta Statistika Informatika. Selain itu saya suka melakukan beberapa aktivitas olahraga di akhir pekan, seperti berlari, bulu tangkis, atau futsal. Saya selalu mencari peluang untuk berkolaborasi dalam proyek-proyek yang berdampak dan inovatif.
